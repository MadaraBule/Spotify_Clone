# Spotify React Web Client

Spotify Web Client

This project uses Bootstrap, HTML, CSS and Javascript.

## Features

- Play full audio tracks.
- Control playback (pause, volume, shuffle, etc).
- Search 
- Log in and Sign Up
- Add or edit your playlists.
- "Responsiveness"
